VTA 配置#

import tvm
from tvm import te
import vta
import numpy as np

函数 vta.get_env() 可以获取 VTA 的参数：

env = vta.get_env()

env.TARGET

'tsim'

用户可以自由地修改影响硬件设计布局的高级硬件参数。这些参数在 tvm/3rdparty/vta-hw/config/vta_config.json 中通过它们的 log2 值指定。

为了便于修改 VTA 硬件参数，可以使用 update_vta_config() 自动修改配置：

from tvm_book.vta.config import update_vta_config

tvm_root = TVM_ROOT
update_vta_config(tvm_root, **{'TARGET': "sim"})

再次查看：

env = vta.get_env()

env.TARGET

'sim'

备注

一定要重写导入 vta，才可以生效。

VTA 多环境管理#

env = vta.get_env()
mock = env.mock
mock.alu

'skip_alu'

借助 with 管理 VTA 环境：

def test_env_scope():
    env = vta.get_env()
    cfg = env.cfg_dict
    cfg["TARGET"] = "xyz"
    with vta.Environment(cfg):
        assert vta.get_env().TARGET == "xyz"
    assert vta.get_env().TARGET == env.TARGET

test_env_scope()
