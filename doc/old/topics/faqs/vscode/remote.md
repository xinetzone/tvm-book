# VSCode Remote 开发 FAQ

```{note} VSCode 远程开发
[Visual Studio Code 远程开发](https://code.visualstudio.com/docs/remote/faq#remote-articles) 扩展包允许您在容器中、远程机器（通过 SSH）或 Windows 子系统（WSL）中打开任何文件夹，并充分利用 VS Code 的全部功能集。这意味着无论您的代码位于何处或托管在哪里，VS Code 都可以提供本地质量的开发生态——包括完整的 IntelliSense（自动完成）、调试等。
```

```{note} 远程开发扩展是如何工作的？
Visual Studio Code 远程开发允许您的本地 VS Code 安装通过将某些命令的执行移动到“远程服务器”，透明地与其他机器（无论是虚拟的还是物理的）上的源代码和运行时环境交互。当您连接到远程端点时，VS Code 会快速安装 VS Code Server，它可以托管直接与远程工作区、机器和文件系统交互的扩展。
```
