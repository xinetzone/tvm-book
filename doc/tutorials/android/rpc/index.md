# Android TVM RPC

参考 [android_rpc](https://github.com/apache/tvm/tree/main/apps/android_rpc) 和 [deploy_model_on_android](https://tvm.apache.org/docs/how_to/deploy_models/deploy_model_on_android.html)

Android RPC 应用程序，它允许我们在 Android 设备上启动 RPC 服务器，并可通过 Python脚本连接至该服务器。之后，我们可以像处理常规 TVM RPC 那样在 Python 端进行测试。
