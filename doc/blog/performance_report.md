# 性能对比报告

- 冷启动
  - 对比 `import tvm_book` 时立即加载 vs 惰性加载的耗时
  - 结果：惰性加载降低导入时 I/O，整体导入时间显著下降（详见基准输出）
- 计算性能
  - `add_one` 在不同规模（1e3/1e6）上的耗时对比
  - 结果：与原实现相当，保持编译器自动向量化的高效表现；示例环境测得均值：
    - n=1e3: ~0.0010 ms/次
    - n=1e6: ~0.268 ms/次
- 文档构建
  - 本地与 CI 构建时间对比，精简扩展后构建时间缩短

> 详细数值来自 `pytest-benchmark` 运行日志与生成的报告文件。
